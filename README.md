# harborctl

## How to Install

```bash
# [install]
brew tap ghdwlsgur/harborctl
brew install harborctl

# [upgrade]
brew tap ghdwlsgur/harborctl
brew upgrade harborctl

# latest version (2023.12.30)
$ harborctl --version
harborctl version 1.0.1

# [refresh]
brew update --verbose
```

## Release

```bash
make release TAG=v1.0.1
```
